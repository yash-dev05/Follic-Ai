## рџ§  How Follic AI Works

Workflow:

1. User uploads hair image
2. Image sent to FastAPI backend
3. AI detects face and hair region
4. Hair density scoring algorithm runs
5. Questionnaire data improves personalization
6. Results stored in database
7. Dashboard displays insights and recommendations
